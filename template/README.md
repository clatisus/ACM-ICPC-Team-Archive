# 模版目录说明

按照下面的目录结构放置模版文件，每个文件至少需要两个人确认，然后标记删除线。

每个模版都应该通过题目检验，统一代码风格，带有清空函数。

模版最好都参考 tls 和昂神的板子，在此基础上增删内容。

括号内部为写者，写者自己找人验板子。
